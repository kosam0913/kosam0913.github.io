隐私政策

Privacy Policy

1. 信息收集

1. Information Collection

TreeNode作为Chrome浏览器插件，不会收集或存储任何用户数据。所有数据都直接从用户的Notion账户获取并实时显示。我们不会收集任何个人信息或Notion数据。

As a Chrome browser extension, TreeNode does not collect or store any user data. All data is retrieved directly from the user's Notion account and displayed in real-time. We do not collect any personal information or Notion data.

2. 数据安全

2. Data Security

TreeNode采用安全的Notion API进行数据访问，所有数据都直接在用户的浏览器中处理。我们不会在服务器端存储任何数据，也不会将数据传输到任何第三方服务器。

TreeNode uses secure Notion API for data access, and all data is processed directly in the user's browser. We do not store any data on the server side, nor do we transmit data to any third-party servers.

3. 数据访问

3. Data Access

TreeNode仅访问用户授权的Notion任务数据库，用于显示树形视图。我们不会访问或修改用户的任何其他Notion数据。

TreeNode only accesses the Notion task database authorized by the user for displaying the tree view. We do not access or modify any other Notion data of the user.

4. 用户控制

4. User Control

用户可以随时通过Chrome扩展管理界面禁用或卸载TreeNode。禁用或卸载后，插件将立即停止访问Notion数据。

Users can disable or uninstall TreeNode at any time through the Chrome extension management interface. After disabling or uninstalling, the extension will immediately stop accessing Notion data.

5. 政策更新

5. Policy Updates

本隐私政策可能会不时更新，我们将通过Chrome Web Store或官方网站发布更新版本。

This privacy policy may be updated from time to time. We will post updated versions through the Chrome Web Store or official website.

If you have any questions or concerns, please contact us at shirantech913@gmail.com.
