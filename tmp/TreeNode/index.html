TreeNode - Notion Task Tree View

可视化你的Notion数据库任务为树状结构

Visualize your Notion database tasks as a tree structure

TreeNode是一个Chrome浏览器扩展，它能将您的Notion数据库任务以树形视图的方式展示，帮助您更直观地理解和管理任务的层级关系，提高工作效率。

TreeNode is a Chrome browser extension that displays your Notion database tasks in a tree view, helping you more intuitively understand and manage task hierarchies, improving your productivity.

📖 使用指南

📖 Usage Guide

1. 连接到Notion

安装扩展后，点击Chrome工具栏中的扩展图标，然后点击"连接到Notion"。在弹出窗口中授权应用程序。

2. 查看任务树

访问任何Notion数据库页面，点击扩展图标查看任务树状图。任务会根据父子关系自动布局。

3. 刷新数据

数据会在每次打开插件时自动刷新，确保您始终看到最新的任务状态。

1. Connect to Notion

After installing the extension, click the extension icon in the Chrome toolbar, then click "Connect to Notion". Authorize the application in the pop-up window.

2. View Task Tree

Visit any Notion database page, click the extension icon to view the task tree. Tasks will be automatically arranged according to parent-child relationships.

3. Refresh Data

Data will be automatically refreshed each time you open the plugin, ensuring you always see the latest task status.

🔗 前提条件

🔗 Prerequisites

要使用TreeNode，您需要：

  • 一个包含任务的Notion数据库
  • 数据库需要有以下字段:
    • 标题 (Title)
    • 状态 (Status): 用于跟踪任务进度
    • 父任务 (Parent task): 关联类型字段，用于建立任务间的层级关系
    • 子任务 (Sub-task): 关联类型字段，用于建立任务间的层级关系

To use TreeNode, you need:

  • A Notion database containing tasks
  • The database needs to have the following fields:
    • Title
    • Status: For tracking task progress
    • Parent task: Relation type field, used to establish hierarchical relationships between tasks
    • Sub-task: Relation type field, used to establish hierarchical relationships between tasks
