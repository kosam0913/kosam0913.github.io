1. Connect to Notion

After installing the extension, click the extension icon in the Chrome toolbar, then click "Connect to Notion". Authorize the application in the pop-up window.

2. View Task Tree

Visit any Notion database page, click the extension icon to view the task tree. Tasks will be automatically arranged according to parent-child relationships.

3. Refresh Data

Data will be automatically refreshed each time you open the plugin, ensuring you always see the latest task status.
